# S01-01 Banners do topo rotativos

Status: concluido
Frente: frontend
Prioridade: P0
Atualizado: 17/06/2026 BRT

## Descricao
Transformar a area de banners do topo em carrossel rotativo (auto-rotacao), com 3 ou mais banners de tamanhos/estrategias diferentes. Cada banner trabalha uma estrategia (campanha, linha ecologica, oferta). Client component por causa da rotacao.

## Checklist
- [x] Carrossel com auto-rotacao e troca suave
- [x] 3+ banners com tamanhos/estrategias variados
- [x] Indicadores (dots) e controles (anterior/proximo)
- [x] Pausa no hover, acessivel por teclado
- [x] Cada banner leva a um destino (catalogo/campanha)
